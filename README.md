# 🚀 TeamFlow✌️ (Frontend)

## 📖 Git Flow 규칙

이 프로젝트는 **Git Flow** 전략을 따르며, 브랜치는 다음과 같이 운영됩니다!

### 🔹 브랜치 전략

| 브랜치     | 설명                              |
| ---------- | --------------------------------- |
| `main`     | 운영(배포) 브랜치                 |
| `develop`  | 개발 브랜치 (기능 개발 및 테스트) |
| `feat/*`   | 새로운 기능 개발 브랜치           |
| `hotfix/*` | 운영 중 긴급 수정 브랜치          |
| `design/*` | 디자인 및 스타일 브랜치           |

---

### 🔹 설치 코드 메모리

| CMD 실행 코드                                      | 설명                           |
| -------------------------------------------------- | ------------------------------ |
| `npx create-react-app . --template typescript`     | react typeScript 프로젝트 생성 |
| `npx tsc --init`                                   | tsconfig.json 생성             |
| ` npm i @types/react @types/react-dom`             | 리액트 돔 설치                 |
| `npm i react-router-dom@6`                         | 리액트 라우터 설치             |
| `npm i styled-components sass`                     | styled-components 설치         |
| `npm i socket.io-client bootstrap react-bootstrap` | 소켓관련 설치                  |
| `npm i redux react-redux @reduxjs/toolkit`         | redux모듈 설치                 |
